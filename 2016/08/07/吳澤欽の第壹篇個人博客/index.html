吳澤欽の第壹篇個人博客

分享下怎麼建立自己的個人博客

安裝

工欲善其事 必先利其器

1、Node.js

Node.js下載 v4.4.7 LTS

2、Git

由於本人使用的是Mac 在終端自帶Git

Windows的同學 Git下載

3、Hexo

在終端用命令行安裝Hexo需確保安裝Command Lind Tools
在終端輸下述命令即可

$ sudo npm install –unsafe-perm –verbose -g hexo

$ cd xxx

$ hexo init

$ sudo npm install

postscript：妳可以使用簡寫命令 $ hexo g $ hexo s 代替上述命令
妳可以看下妳目前的成果了 本地服務器查看

服務器

把妳的博客託管到Github上面是個很不錯的選擇

如果沒有就去註冊Github帳號
創建一個倉庫用於存放博客 ⚠️xxx.github.io xxx必須是妳GitHub的用戶名

接著把妳的Hexo部署到Github
編輯在Hexo下的_config.yml文件 ⚠️用戶名的替換

然後執行

F1G1NS1.DNSPOD.NET

F1G1NS2.DNSPOD.NET
```

大功告成！

後續

萬里長征第一步

擁有自己的博客網站後需要用心的去打理
妳可能還需要用 圖床 來上傳圖片
需要更換博客的主題或者自己設計博客
總之妳已經邁出第一步了那就堅持走下去吧 _(:」∠)_
